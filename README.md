# ServiFix
ServiFix es una empresa especializada en el soporte, mantenimiento y gestión operativa de la infraestructura eléctrica utilizada por compañías del sector energético. Dentro de los principales activos que atiende se encuentran postes de iluminación, transformadores, líneas de transmisión y distribución, entre otros componentes críticos del sistema eléctrico.
El proyecto consiste en implementar una plataforma que centraliza la gestión de incidentes provenientes de un sistema de monitoreo externo. Este sistema genera tickets cuando se detecta alguna anomalía en los componentes eléctricos, los cuales son asignados automáticamente a un líder técnico junto con un código de incidente para su diagnóstico y atención durante la jornada.
Toda la información es almacenada en una base de datos y, de manera diaria, los registros se exportan en formato .csv hacia un contenedor en Azure. Adicionalmente, el sistema integra un archivo maestro que define los niveles de criticidad por zona, utilizando un esquema tipo semáforo (Alta, Media, Baja). Este nivel se determina comparando el tiempo transcurrido desde la asignación de la cuadrilla hasta el tiempo máximo esperado definido en el maestro. El proceso puede ejecutarse múltiples veces al día, actualizando en tiempo real la base de datos.
Los datos consolidados se presentan en un dashboard en Databricks, permitiendo a los gerentes visualizar el comportamiento de los incidentes, analizar estadísticas operativas y tomar decisiones informadas para optimizar la gestión y mejorar la eficiencia del servicio
